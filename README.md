## Description

Twitter clone frontend app built with [Next.js](https://nextjs.org/).

## Getting Started

Always remember to install node modules

```bash
npm install
```

Running the app

```bash
# Production
npm run start

# Development
npm run dev
# or
yarn dev

# Testing (coming soon)
npm run test
```
